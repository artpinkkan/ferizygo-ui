# FerizyGo

**FerizyGo** is a frontend web application for a premier Indonesian ferry booking platform. The platform is designed to provide users with a seamless sea travel experience, allowing easy booking for passengers and vehicles across various popular routes in Indonesia (such as Merak to Bakauheni, Ketapang to Gilimanuk, and more).

## Project Overview

This project focuses on the user interface and frontend interactions for ferry and sea travel bookings. It includes responsive designs and interactive elements for route searching, ticket booking, and user authentication.

## Key Features

- **Route Search & Booking**: Users can select origin and destination ports, departure dates, number of passengers, and vehicle types.
- **Popular Routes Showcase**: Quick access to frequently traveled routes with estimated travel times and starting prices.
- **Authentication**: Built-in UI for user login and signup modals.
- **Responsive Navigation**: A clean and accessible navigation system for routes, bookings, and support.

## Pages & Structure

- `landing.html` - The main entry point featuring the hero section, booking widget, features (Real-time tracking, fast checkout, transparent pricing), and popular routes.
- `search-routes-detail.html` - Detailed view of available ferry routes matching the user's search criteria.
- `checkout.html` - The checkout process for finalizing ticket purchases.
- `schedule.html` - Ferry schedules and timetables.
- `sign-in.html` / `sign-up.html` - Dedicated pages/modals for user account management.
- Additional CSS (`landing.css`, `checkout.css`, `auth.css`, etc.) and JavaScript (`landing.js`, `checkout.js`, etc.) files for styling and interactivity.

## Technologies Used

- **HTML5**: Semantic structure for all pages.
- **CSS3**: Custom vanilla CSS for styling, layout, and responsive design (no external CSS frameworks).
- **JavaScript (Vanilla)**: For handling UI interactions, modals, and dynamic content.

## How to Run

1. Clone or download the repository to your local machine.
2. Open `landing.html` in your preferred web browser to explore the application.
3. Navigate through the site using the provided links and features to experience the booking flow.
